# Banana Drum

Music sequencer web-app designed for samba bands

Check it out! [bananadrum.net](https://bananadrum.net)

## Setup

```
npm install
```

## Building

Compile TypeScript and bundle with webpack:

```
npm run build
```

## Running

```
npm start
```

To have hot reload run this in another terminal:
```
tsc --build --watch
```

## Testing

```
npm test
```
